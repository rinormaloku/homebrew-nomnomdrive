# homebrew-nomnomdrive

Homebrew Cask tap for [NomNomDrive](https://github.com/rinormaloku/nomnomdrive) — local AI-powered document search and chat.

## Install

```sh
brew tap rinormaloku/nomnomdrive
brew install --cask nomnomdrive
```

## Upgrade

```sh
brew upgrade --cask nomnomdrive
```

## Uninstall

```sh
brew uninstall --cask nomnomdrive
```
